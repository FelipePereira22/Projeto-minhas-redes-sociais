# Link-in-Bio / Agregador de Redes Sociais 🌐

> 🔗 **[Clique aqui para acessar o projeto no ar](https://felipepereira22.github.io/Projeto-minhas-redes-sociais/)**

Uma aplicação web estilo "Linktree" desenvolvida para centralizar e apresentar de forma elegante minhas principais redes sociais, portfólio e canais de contato. O projeto foca em uma interface minimalista e de alta legibilidade.

## 📸 Demonstração da Interface

A interface foi projetada com um tema escuro (*dark mode*) sofisticado, contendo um card centralizado com foto de perfil, subtítulo profissional e botões interativos de redirecionamento.

## 🚀 Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica da página, utilizando tags adequadas para acessibilidade e SEO.
*   **CSS3:** Estilização moderna aplicada ao layout, incluindo:
    *   Uso avançado de **Flexbox** para centralização vertical e horizontal perfeita do card.
    *   Efeitos de **Hover** e transições suaves nos botões para melhorar a experiência do usuário (*UX*).
    *   **Design Responsivo:** Adaptado perfeitamente para telas de smartphones e computadores através de Media Queries.

## 💡 Aprendizados e Conceitos Praticados

Este projeto, apesar de conceitualmente simples, permitiu consolidar boas práticas de Front-End, tais como:
*   Alinhamento rigoroso de elementos na tela (um dos desafios mais comuns de CSS).
*   Gerenciamento de espaçamentos (`padding` e `margin`) para criar uma interface limpa e simétrica.
*   Uso de propriedades de borda, sombras leves e paleta de cores harmoniosa.
